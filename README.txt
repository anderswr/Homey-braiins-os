Braiins OS+ Miner Controller

Control and monitor Antminer devices running Braiins OS+ firmware.

Features:
- Pause / Resume mining
- Per-device configuration (IP, port, HTTPS, username, password)
- Power consumption monitoring (W)
- Secure connection test during setup

Requirements:
- Antminer running Braiins OS+ firmware
- Homey Pro (2019 or newer)
- Miner must be on the same local network as Homey

Setup:
1. Add device from the app
2. Enter miner IP, port, and credentials
3. Test connection
4. Control mining via the On/Off toggle

This is an early test version intended for LAN usage.
